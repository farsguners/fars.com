# $FARS Token Website - Development Plan

## Design Guidelines

### Design References
- **Web3/Crypto sites**: Dark futuristic aesthetic with neon accents
- **Football fan culture**: Stadium energy, community passion
- **Style**: Futuristic Dark Mode + Neon Glow + Glassmorphism

### Color Palette
- Background Primary: #0a0a1a (Deep Navy Black)
- Background Secondary: #0d1117 (Dark Navy)
- Card Background: rgba(255, 255, 255, 0.05) (Glassmorphism)
- Accent Red: #ff2d55 (Neon Red)
- Accent Blue: #007aff (Electric Blue)
- Gradient: linear-gradient(135deg, #ff2d55, #007aff)
- Text Primary: #ffffff
- Text Secondary: #a0a0b0

### Typography
- Headings: Orbitron (futuristic, bold)
- Body: Poppins (clean, modern)

### Key Component Styles
- Buttons: Gradient red→blue, neon glow on hover
- Cards: Glassmorphism with subtle border glow
- Navbar: Fixed, transparent with blur backdrop
- Sections: Fade-in on scroll animations

### Images to Generate
1. **logo-fars-token.png** - Futuristic fan token logo, red and blue neon glow, cannon symbol, dark background (Style: minimalist, transparent background)
2. **hero-stadium-lights.jpg** - Futuristic stadium with neon lights, dark atmosphere, red and blue lighting, Web3 aesthetic (Style: photorealistic, cinematic)
3. **background-grid-dark.jpg** - Dark futuristic grid pattern with subtle red and blue glow lines, abstract tech background (Style: photorealistic, dark mood)
4. **team-placeholder-avatar.jpg** - Futuristic silhouette avatar, dark background with neon outline, Web3 style (Style: minimalist, dark)

## File Structure
- index.html - Homepage (Hero, About, Vision, Features, Footer)
- tokenomics.html - Tokenomics page with pie chart
- roadmap.html - Roadmap timeline Q3 2026 - Q3 2027
- team.html - Team member cards
- partners.html - Partners logo grid
- style.css - All styling (dark mode, animations, responsive)
- script.js - 3D particles, scroll animations, navbar, pie chart

## Development Tasks
1. Generate all images using ImageCreator
2. Write style.css - Complete dark mode styling with animations
3. Write script.js - Three.js particles, scroll animations, pie chart
4. Write index.html - Homepage with all sections
5. Write tokenomics.html - Token distribution page
6. Write roadmap.html - Timeline page
7. Write team.html - Team cards page
8. Write partners.html - Partners grid page
